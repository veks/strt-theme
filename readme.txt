Минималистичная блочная тема WordPress для быстрого старта разработки сайтов.

== Description ==

**STRT** — это легкая и быстрая блочная тема для WordPress, созданная как стартовая основа для разработки. Идеально подходит для разработчиков, агентств и всех, кто хочет начать с чистого, не перегруженного шаблона. Тема полностью поддерживает редактор сайта (Full Site Editing), шаблоны, стили, кастомизацию блоков и адаптивный дизайн.

Особенности:
- Чистая структура
- Полная поддержка FSE (редактор сайта)
- Минималистичный дизайн без лишнего кода
- Поддержка кастомных шаблонов и стилей

= Instructions =

1. Загрузите архив темы через админку WordPress или распакуйте в папку `/wp-content/themes/`.
2. Активируйте тему через меню **Внешний вид > Темы**.
3. Перейдите в редактор сайта (**Внешний вид > Редактор**) для настройки шаблонов и стилей.
